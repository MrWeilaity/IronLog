## [1.0.0] - 2025-12-15

### 新增

#### 核心功能
- 🏋️ **训练管理系统**
  - 训练记录功能（记录动作、组数、重量、次数、RPE）
  - 基础动作库（15个预置动作）
  - 训练计划功能（周计划制定）
  
- 🍎 **营养管理系统**
  - 饮食记录功能（按餐次记录）
  - 食物营养库（10个预置食物）
  - 营养素自动计算（热量、蛋白质、碳水、脂肪）
  
- 💧 **饮水记录**
  - 每日饮水量记录
  - 饮水目标设置
  - 饮水进度显示

- 📊 **数据分析**
  - 仪表盘数据展示
  - 体重趋势图
  - 训练容量统计
  - 营养摄入分析

- 📅 **日历功能**
  - 月视图日历
  - 训练打卡显示
  - 日期快速导航

- 👥 **用户系统**
  - 用户注册和登录
  - 个人资料管理
  - 密码修改功能
  - 角色权限控制（USER/ADMIN）

- 🔧 **后台管理**
  - 用户管理（查看、禁用）
  - 系统公告发布
  - 系统设置管理

#### 技术实现

##### 后端
- Spring Boot 3.1.5 框架
- Spring Data JPA 数据访问
- MySQL 8.0 数据库
- RESTful API 设计
- 密码 SHA-256 加密

##### 前端
- Vue 3.3.4 框架
- Element Plus 2.4.2 UI 组件库
- ECharts 5.4.3 数据可视化
- Vue Router 4.2.5 路由管理
- Vite 4.5.0 构建工具

##### 数据库
- 9 张核心业务表
  - sys_user（用户表）
  - train_plan（训练计划）
  - train_record（训练记录）
  - base_action（动作库）
  - food_nutrition（食物营养库）
  - diet_log（饮食记录）
  - water_log（饮水记录）
  - system_announcement（系统公告）
  - system_settings（系统设置）

### 初始数据

## 版本说明

### 版本号规则

版本格式：主版本号.次版本号.修订号 (MAJOR.MINOR.PATCH)

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 变更类型

- `新增` - 新功能
- `变更` - 现有功能的变更
- `弃用` - 即将移除的功能
- `移除` - 已移除的功能
- `修复` - Bug 修复
- `安全` - 安全问题修复
