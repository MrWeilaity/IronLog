# IronLog 界面设计指南

本文档提供 IronLog 系统各页面的界面布局说明和设计规范。

## 📱 总体设计理念

- **简洁明了**: 界面清爽，信息层次分明
- **数据驱动**: 核心数据醒目展示，一目了然
- **操作便捷**: 常用功能快速可达，减少点击次数
- **视觉反馈**: 操作实时反馈，状态清晰可见

## 🎨 色彩方案

### 主色调

```
主色: #409EFF (Element Plus 蓝色)
成功: #67C23A (绿色)
警告: #E6A23C (橙色)
危险: #F56C6C (红色)
信息: #909399 (灰色)
```

### 应用场景

- **蓝色**: 主要按钮、链接、进度条
- **绿色**: 成功提示、完成状态、正向指标
- **橙色**: 警告信息、接近目标
- **红色**: 错误提示、超标、删除操作
- **灰色**: 次要信息、禁用状态

## 📐 主要页面布局

详细的界面布局和交互说明，请参考 PROJECT_DOCS.md 的用户操作手册部分。

### 登录页面
- 居中卡片式布局
- 用户名和密码输入框
- 登录和注册按钮
- 测试账号提示

### 仪表盘
- 营养摄入卡片（4个进度条）
- 饮水打卡区
- 训练计划提醒
- 系统公告列表

### 训练日记
- 日期选择器
- 训练概况卡片
- 动作列表（可展开）
- 每组数据录入表格

### 饮食记录
- 日期选择器
- 营养总览（环形图）
- 按餐次分组（早中晚加餐）
- 食物列表和营养素显示

### 数据统计
- 体重趋势折线图
- 训练容量柱状图
- 肌群分布饼图
- 月度统计卡片

### 我的日历
- 月度日历视图
- 训练打卡标记
- 月度统计数据

## 🎭 交互规范

### 按钮状态
- **默认**: 蓝色背景 (#409EFF)
- **悬停**: 浅蓝色 (#66B1FF)
- **激活**: 深蓝色 (#3A8EE6)
- **禁用**: 灰色 (#A0CFFF)

### 消息提示
- **成功**: 绿色 ✅
- **警告**: 橙色 ⚠️
- **错误**: 红色 ❌
- **信息**: 蓝色 ℹ️

## 📱 响应式设计

- **桌面端** (≥1200px): 侧边栏展开
- **平板端** (768-1199px): 侧边栏可收起
- **移动端** (<768px): 顶部菜单 + 底部导航

## 🎨 组件使用

### Element Plus 核心组件
- Container, Menu, Form, Table, Card
- Button, Input, Select, DatePicker
- Message, Dialog, Progress

### ECharts 图表
- 折线图: 体重/容量趋势
- 柱状图: 训练统计
- 饼图: 肌群分布
- 环形图: 营养比例
